SkyExplorer AI is a high-performance web application that integrates professional astronomical mapping with generative artificial intelligence. The platform leverages the Aladin Lite v3 API to provide an interactive visualization of the Sloan Digital Sky Survey (SDSS). By utilizing Google Gemini 3.1 Flash, the system performs real-time scientific analysis of celestial objects. A critical innovation of the project is the implementation of a persistent caching layer using Supabase (PostgreSQL), which indexes AI-generated summaries by celestial coordinates to optimize performance and minimize API latency.
